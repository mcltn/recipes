# My Recipes

A simple, beautiful GitHub Pages site to keep track of your favorite recipes using markdown files.

## Features

- 📝 Create recipes using markdown with structured frontmatter
- 🏷️ Organize recipes by category (breakfast, lunch, dinner, dessert, etc.)
- 📄 Each recipe is a version-controlled markdown file
- 💾 Recipes are backed up in your Git repository
- 🔗 Each recipe gets its own dedicated page with a clean design
- 📱 Responsive design that works on mobile and desktop

## Getting Started

### Local Development

1. Create a new GitHub repository named `recipes`
2. Clone it to your computer
3. Copy all files from this folder into your repository

### Creating Recipes

Create a new markdown file in the `recipes/` folder with the following format:

```markdown
---
layout: recipe
title: Recipe Name
category: Breakfast
prep_time: 15 minutes
cook_time: 30 minutes
servings: 4
ingredients:
  - 2 cups flour
  - 1 cup sugar
  - 3 eggs
---

1. Step one
2. Step two
3. Step three
```

**Frontmatter fields:**
- `layout: recipe` - Required, tells Jekyll to use the recipe layout
- `title` - Recipe name
- `category` - Category (Breakfast, Lunch, Dinner, Dessert, Snack, Beverage, etc.)
- `prep_time` - Optional preparation time
- `cook_time` - Optional cooking time
- `servings` - Optional number of servings
- `ingredients` - List of ingredients (YAML array format)

**Content:** Write the cooking instructions below the frontmatter (steps, notes, etc.)

### Deploy to GitHub Pages

1. Push to GitHub:
   ```bash
   git add .
   git commit -m "Initial commit: Add recipe site"
   git push origin main
   ```

2. In your GitHub repository settings:
   - Go to **Settings** → **Pages**
   - Select **Source**: main branch
   - Click **Save**

3. Your site will be available at `https://yourusername.github.io/recipes`

Jekyll will automatically build the site and render all markdown files.

## File Structure

```
recipes/
├── _config.yml              # Jekyll configuration
├── _layouts/
│   ├── default.html        # Main layout
│   └── recipe.html         # Recipe page layout
├── style.css               # Styling
├── index.md                # Homepage listing all recipes
├── recipes/
│   ├── chocolate-chip-cookies.md
│   ├── scrambled-eggs.md
│   └── your-recipe.md
└── README.md
```

## Notes

- Recipes are stored as git-tracked files, so they're fully backed up
- The homepage automatically lists all recipes grouped by category
- Each recipe gets its own unique page with the recipe title in the browser tab
- No database or backend needed - it's all static HTML generated by Jekyll

## Customization

Feel free to modify:
- **Colors**: Edit the gradient colors in `style.css`
- **Layout**: Edit `_layouts/recipe.html` for individual recipe pages
- **Theme**: Change `theme` in `_config.yml` to use a different Jekyll theme

Enjoy keeping track of your favorite recipes! 🍳
